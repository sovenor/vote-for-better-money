# Project Brief — Vote For Better Money

## Overview
"Vote For Better Money" (voteforbetter.money) is an educational website that helps Americans understand why Bitcoin is better money than the US dollar. The core thesis: inflation and money printing erode the dollar's value, while Bitcoin's fixed supply of 21 million offers a superior alternative that anyone can start using today — no election required.

## Core Requirements
- **Educate:** Explain inflation, money printing, national debt, and how Bitcoin solves these problems
- **Persuade:** Use real-time economic data (FRED, BTC price) to make the case visually compelling
- **Activate:** Guide visitors to start using Bitcoin (buy, earn, self-custody)
- **Reach:** SEO-optimized, mobile-friendly, fast-loading, dark-themed

## Target Audience
Everyday Americans who are frustrated with inflation and rising prices but may not yet understand Bitcoin. The tone is accessible, non-technical, and politically neutral (both parties are criticized for debt/inflation).

## Key Pages
1. **Homepage** — Hero + stats + problem statement + Bitcoin as solution + freedom cards + CTAs
2. **Learn section** — 5 articles: Bitcoin is American, Sovereign, Scarce, Decentralized, Permissionless
3. **How to Use Bitcoin** — Practical guide to buying, earning, and self-custodying Bitcoin
4. **Flyers** — Downloadable PDF flyer for grassroots distribution
5. **Links** — Shareable link page for social media
6. **Stats API** — Real-time data from FRED, TwelveData, and Mempool.space
